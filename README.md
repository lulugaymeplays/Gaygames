# Slingshot Mayhem - Web Game

## Como Deployar
Como os arquivos já estão na pasta `C:\Users\luiza.castro_uscsonl\Documents\Gaygames`:
1. Abra o **GitHub Desktop**.
2. O repositório detectará os novos arquivos.
3. Faça o **Commit** e o **Push**.
4. No GitHub, ative o **GitHub Pages** nas configurações do repositório.

## Comandos do Jogo
- **Menu**: Selecione o número de jogadores.
- **Jogo**: No seu turno (mostrado no topo), clique e arraste seu personagem na direção OPRESTA ao alvo (como um estilingue) e solte para atirar.
- **Objetivo**: Destrua o terreno ou atinja os inimigos para derrubá-los ou zerar a vida deles.
